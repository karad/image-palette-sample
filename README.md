# image-palette-sample

Make color palette from image.

## Setup

1. Clone this repo.
2. Change directory to this repo `cd image-palette-sample/`
3. Run `npm i`

## Usage

Execute command on terminal.

```
$ npm run palette /YOUR/IMAGE/PATH/
```

ex.

```
$ npm run palette ./test.jpg

Vibrant : #e22584
DarkVibrant : #24b4bb
LightVibrant : #e27db1
Muted : #b07190
DarkMuted : #3a5557
LightMuted : #d4a4bc
```
